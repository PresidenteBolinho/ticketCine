# ticketCine
Aplicação para venda de ingressos de cinema

# Sobre

É um projeto recomendado para fazer em live (https://www.twitch.tv/presidentebolinho), tem o objetivo de criar uma aplicação para venda de incressos de cinema

# Requisitos

## Interface de usuario

### Home
* Barra com ingresso, Pesquisa, localização e login
* Sumario com categorias dos filmes
* Banners dos filmes
    * Em alta
    * Em cartaz
    * Em breve

### Autenticação
* Login
* Cadastro

### Salas de cinema
* Cadeiras

### Fechamento da compra
- Ingressos
    * Cinema
    * Filme
    * Tipo sessão (2D/3D)
    * Horario
    * Quantidade
    * Preço unitário
    * Preço total
- Pagamento
    * Cartão de crédito
    * Paypal
    * Pix
    * Cupom de desconto

### Area do usuario
* Feed (Ingressos validos, vencidos, consumidos)
* Configurações


## Banco de dados

### Cadastro
* id
* Usuario
* Senha
* Email
* Data de nascimento
* Data criação


### Filmes
* Nome
* Sinopse
* Faixa Etaria
* Preço
* Sessões
